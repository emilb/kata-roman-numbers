# A Kata for converting integers to Roman numerals

## Pre'reqs

- Node, npm

## Running

```bash
> npm install
> node index.js 23 03 1763
23 : XXIII
03 : III
1763 : MDCCLXIII
```

## npm scripts

- lint
  - runs the linter (use --fix to autocorrect)
- test
  - runs the tests with jest
- watch
  - runs jest in watch-mode
- mutate
  - runs a mutation test
